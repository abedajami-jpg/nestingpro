FEATURES

Everything You Need to Cut Smarter

Designed for fiber laser operators — no CAM experience needed.

📂
DXF Import
Import any DXF file from AutoCAD or any CAD software. Supports nested blocks, LWPOLYLINE, circles, and arcs.
🔲
Auto-Nesting
Automatically arrange multiple parts on the sheet with configurable gap, margin, and start position. Minimize material waste.
⚡
PRG Generation
Generate LYNCA CNC controller PRG files instantly. Correct G-codes, pierce points, lead-ins, and G827 lead-outs.
🖱️
Interactive Preview
Click and drag parts to reposition. Rotate parts 90°. Zoom and pan the sheet preview. What you see is what gets cut.
✂️
Line Cutting Mode
Auto-detects strip/slitting files. Filters boundary lines, generates correct pierce and lead-out for each line. Click to mark done.
📌
Unitary Mode
Use exact DXF coordinates without nesting. Perfect for single parts already positioned for the machine origin.
🎯
Machine Origin
Select your machine origin (Top-Left, Bottom-Left, Top-Right, Bottom-Right). Coordinates auto-converted in PRG output.
⏱️
Cut Time Estimation
Accurate time estimates based on 2kW fiber laser speed tables for Steel, Stainless, Aluminum, Galvanized per thickness.
💾
100% Offline
Single HTML file. No installation, no internet, no cloud. Works on any laptop. Share via USB or email. Auto-incremental file naming.
HOW IT WORKS

From DXF to Cutting in 4 Steps

NestingPro simplifies your workflow from drawing to finished parts.

1
Import DXF
Drop your AutoCAD DXF file. Choose Nesting, Unitary, or Line Cutting mode.
2
Set Quantities
Set how many of each part you need. Add multiple different parts to one sheet.
3
Auto-Nest
Click Auto-Nest. Parts arranged automatically. Drag to fine-tune positions or rotate.
4
Generate & Cut
Generate PRG file. Download as 001.prg. Load on LYNCA controller and cut!
